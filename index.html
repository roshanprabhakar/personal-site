abstract Protograph-based LDPC codes are a class of high performance linear error-correction codes on sparse bipartite graphs providing efficient decoding using message passing (between variable and check nodes on the graph). My group’s work provides an API to handle these codes, built upon Prof. Radford Neal’s library for the handling of regular LDPC codes.
Functionality included for:
- Generation of regular and protograph LDPC matrices using PEG and library custom construction methods.
- Encoding and decoding of these codes, including support for puncturing.
- Utility and test scripts for analyzing these codes.

Facial-centric Image Compression
affiliation Information Systems Laboratory, Stanford Compression Forum, STEM to SHTEM, Stanford University
title Research/Project Intern
timeline June 2019 - August 2019
advisors Prof. Tsachy Weissman, EE; Kedar Tatwawadi, PhD student; Sarah Abdali, Undergrad
summary Collaborated with high school peers to develop a novel compression algorithm for facial images. Continued this project individually upon the program’s end.
publication
  • The Informaticists Scientific Journal - https://theinformaticists.com/2019/08/28/building-a-human-centric-lossy-compressor-for-facial-images/
abstract This project capitalizes on the idea that the eyes, nose, and mouth are of utmost importance for identity preservation in facial images. The initial implementation posed a rescaling compression of the image alongside full preservation of facial features. Following the STEM to SHTEM program’s conclusion, I developed k-means clustering based jpeg compression for the purposes of edge preservation in the resultant image, aiming to improve the existing schema. Multi-level compression was later implemented: the image background, facial background, and features being subjected to different degrees of compression.
